# 🌿 Еко-Блум — Екологічне Коло Венна

Інтерактивна екологічна гра для учнів 6 класу на основі методу "Кола Венна".

## 🎮 Можливості

- **6 готових тем** — вирубка лісів, батарейки, мода, транспорт, природа-людина-технології, відновлювана енергія
- **2 і 3 кола Венна** — залежно від теми
- **3 режими гри** — Drag & Drop, текстовий ввід, або обидва
- **Теорія** — цікаві факти для кожної теми
- **Система балів** — ⭐ за правильні відповіді
- **Створення тем** — вчитель може додати власні теми
- **Адаптивний дизайн** — працює на мультиборді, ПК та телефоні

## 🚀 Швидкий старт

```bash
npm install
npm run dev
```

## 📦 Збірка для продакшн

```bash
npm run build
```

Готові файли будуть у папці `dist/`.

## 🏫 Для вчителів

1. Відкрийте гру на мультиборді або роздайте посилання учням
2. У **Налаштуваннях** оберіть режим гри
3. Учні обирають тему, читають теорію та грають
4. Можна додавати власні теми через кнопку **➕**

## 🛠️ Технології

- React 18
- Vite
- CSS-in-JS (inline styles)
